# VendCAR - Landing Page

## Descrição
O **VendCAR** é uma landing page desenvolvida com o objetivo de maximizar a conversão de clientes para o setor automotivo. Este projeto foi criado para atender às necessidades de uma agência de marketing, oferecendo uma solução moderna, responsiva e otimizada para atrair e engajar potenciais clientes.

## Funcionalidades
- **Menu Mobile:** Menu responsivo com alternância de ícones e fechamento automático ao clicar fora ou em links.
- **Animações de Rolagem:** Elementos revelados dinamicamente ao rolar a página.
- **Cabeçalho Dinâmico:** Alteração do fundo do cabeçalho ao rolar a página.
- **Rolagem Suave:** Navegação fluida para âncoras internas.
- **Links Ativos:** Destaque automático do link de navegação correspondente à seção visível.
- **Botão de WhatsApp:** Rastreamento de cliques para análise de conversões.
- **Validação de Formulários:** Validação de campos obrigatórios, e-mails e telefones.
- **Lazy Loading:** Carregamento otimizado de imagens para melhorar o desempenho.
- **Animação de Contadores:** Contadores animados para exibição de métricas e estatísticas.
- **Debounce:** Otimização de eventos de rolagem para melhor desempenho.
- **Tela de Carregamento:** Tela opcional para melhorar a experiência do usuário.
- **Botão de Voltar ao Topo:** Acesso rápido ao topo da página.
- **Acessibilidade:** Melhorias como link para pular conteúdo e foco no menu mobile.

## Tecnologias Utilizadas
- **HTML5:** Estrutura semântica e otimizada.
- **CSS3:** Estilização moderna com suporte a variáveis e transições.
- **JavaScript (ES6+):** Funcionalidades interativas e dinâmicas.

## Como Utilizar
1. Clone o repositório:
   ```bash
   git clone https://github.com/Davemottaa/VendCAR.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd VendCAR
   ```
3. Abra o arquivo `index.html` no navegador para visualizar o site.

## Deploy no GitHub Pages
1. Certifique-se de que o repositório está conectado ao GitHub.
2. No GitHub, vá até as configurações do repositório.
3. Na seção **Pages**, configure a branch `main` e o diretório `/root`.
4. O site estará disponível em: `https://<seu-usuario>.github.io/VendCAR/`.
